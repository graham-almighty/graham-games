# Brawl Monsters -- `brawl-monsters/index.html`

## Overview
Monster battler with two modes: a roguelike run (8-floor gauntlet) and a freeplay adventure (2D tile map, leveling, gyms, capture, quests). Single HTML file, no build tools.

## Core Systems
- **Renderer:** DOM-based battle scenes with inline SVG monster sprites; 2D Canvas tile map for freeplay exploration
- **Controls:** Mouse/touch for battles and menus; WASD/Arrow keys for map movement; E to interact; T for team; Q for quests
- **Mobile:** Touch D-pad + action button shown on small screens; responsive CSS breakpoint at 600px
- **Audio:** Web Audio API synthesized SFX (`SFX.hit()`, `SFX.superHit()`, `SFX.miss()`, `SFX.heal()`, `SFX.faint()`, `SFX.select()`, `SFX.victory()`, `SFX.defeat()`, `SFX.buff()`, `SFX.status()`)
- **Fonts:** Cinzel Decorative for titles/buttons, Segoe UI for body text
- **Theme:** Dark background (#0a0a0f), magenta accent (#e040fb)

## Game Modes

### Roguelike Mode (`gameMode = 'roguelike'`)
- **Draft:** 5 random monsters offered (from pool of 13), pick 3 for your team
- **8 floors** (`TOTAL_FLOORS = 8`), each a battle
  - Floor 1: 1 enemy (0.7x scale)
  - Floors 2-3: 2 enemies
  - Floors 4-7: 3 enemies
  - Floor 8: Boss (Chimera Rex)
- **Enemy scaling:** `scale = 1 + (floor - 1) * 0.15` (floor 1 uses 0.7x)
- **Between floors:** full heal + choose 1 of 3 rewards
- **Rewards** (weighted random):
  - ATK Crystal (25w): +10% ATK to all monsters
  - DEF Crystal (25w): +10% DEF to all monsters
  - SPD Crystal (15w): +10% SPD to all monsters
  - HP Crystal (20w): +15 max HP to all monsters
  - Move Tutor (15w): replace one move on one monster with a random learnable move
- **Boss rage:** Chimera Rex gets a second action per turn when below 50% HP
- Victory/defeat tracked in `bm-stats`

### Freeplay Mode (`gameMode = 'freeplay'`)
- **Team selection:** pick 3 from all 13 (or 12 without Drakovex) monsters
- **Checkpoint starts:** if gyms beaten previously, can start at higher levels with pre-assigned moves
- **2D tile map exploration** (30x25 grid, 32px tiles) with zones, NPCs, gym doors
- **Wild encounters:** 3% chance per frame on tall grass (tile type 1), cooldown 60 frames after decline
- **Leveling system:** XP from battles, max level 20
- **Capture system:** after wild battle win, choose one defeated enemy to add to roster (max 20 total)
- **Roster:** 3 active + up to 17 reserves, swap/release from roster screen
- **5 gyms** with leader dialogue, progression badges
- **Quest system:** 10 story quests + 5 side quests from NPCs
- **Save/load:** single freeplay save slot persisted to localStorage

## Combat System

### Damage Formula
```
atkMod = 1 + (attacker.buffs.atk.amount / 100)  // if buffed
defMod = 1 + (defender.buffs.def.amount / 100)  // if buffed
baseDmg = (move.power * (atk * atkMod / def / defMod)) / 4 + 2
finalDmg = floor(baseDmg * effectiveness * variance)  // variance: 0.85-1.0
```
Minimum damage: 1

### Accuracy
```
finalAcc = move.accuracy * (1 + attacker.buffs.accuracy.amount / 100)
hit = random(0-100) < finalAcc
```

### Turn Order
- Priority moves go first
- Otherwise, higher effective speed goes first (speed * speed buff modifier)
- Both combatants act each turn (first attacker, then second if still alive)

### Move Effects
| Effect Type | Description |
|-------------|-------------|
| `stun` | Target skips next turn (chance-based) |
| `debuff` | Reduce target stat for N turns (chance-based) |
| `buffSelf` | Increase own stat for N turns |
| `drain` | Heal attacker by % of damage dealt |
| `healSelf` | Heal % of own max HP |
| `priority` | Always goes first (handled in turn order) |

### AI (`aiSelectAction`)
- Scores each move based on power, effectiveness, accuracy, and effect value
- Considers switching if type advantage available or HP low
- **Smartness scaling:**
  - Roguelike: floor 1 = 0.05, capped at 0.9 (`0.3 + floor * 0.08`)
  - Freeplay wild: capped at 0.7 (`0.3 + avgLevel * 0.02`)
  - Freeplay gym: `0.6 + gymId * 0.06`
- Smartness controls randomness in scoring (lower = more random)

## Freeplay Leveling

### XP Table (`LEVEL_XP`)
```
Lv 1: 0, Lv 2: 100, Lv 3: 250, Lv 4: 450, Lv 5: 700,
Lv 6: 1000, Lv 7: 1400, Lv 8: 1850, Lv 9: 2400, Lv 10: 3000,
Lv 11: 3700, Lv 12: 4500, Lv 13: 5400, Lv 14: 6400, Lv 15: 7500,
Lv 16: 8800, Lv 17: 10200, Lv 18: 11800, Lv 19: 13600, Lv 20: 15600
```

### Stat Scaling
`scale = 1 + (level - 1) * 0.05` applied to base HP/ATK/DEF/SPD

### Battle XP
```
baseXP = sum(30 + enemyLevel * 10 for each enemy)
multiplier = clamp(1.0 + (avgEnemyLvl - avgPlayerLvl) * 0.1, 0.25, 2.0)
finalXP = floor(baseXP * multiplier)
```
- Gym battles: 3x XP multiplier
- Loss: 25% XP awarded

### Freeplay Move Pool (`FREEPLAY_MOVE_POOL`)
Each monster has 2 learnable moves at levels 8 and 15. Prompted when level threshold reached; can replace an existing move or skip (tracked via `_skippedMoves`).

- Walk bob: frame 1 = -1px, frame 2 = +1px vertical offset
- Sprites pre-cached as Image objects for performance

## Market System

- Accessible from map screen via MARKET button
- Buy/Sell tabs
- **Gold** (`fpGold`): earned from battles, displayed on map HUD and market screen
- **Gold earning:** `10 + avgEnemyLevel * 3 + enemyCount * 5` per battle win; gym battles 3x
- **Buy prices:** `floor((hp + atk*2 + def + spd) / 5) + level * 8` — based on base stats + level
- **Sell prices:** 40% of buy price (reserves only, active team cannot be sold)
- Bought monsters go to reserves (max 20 total roster)
- Buy selection shows monsters at levels around player's average (±1)
- Respects Drakovex shop unlock
- Species caught tracking and quest completion checks on purchase

## Battle Features

- **Fullscreen:** toggle button on title screen (`toggleFullscreen()`)
- **Quit battle:** QUIT button in battle top bar, returns to map on loss path
- **Stat scanner** (G Bux item `bm_stat_scanner`): shows enhanced enemy pips with name, HP bar, and element color
- **Enemy trainer:** gym leaders appear behind their monster in battle (positioned at left:12%, opacity 0.6)
- Gym leaders have trainer appearance data in GYMS array

## Tutorial System

7 tutorial steps triggered at specific moments:
1. `draft` -- Build Your Team (shown on draft screen)
2. `draft-select-1` -- Elements Matter (after first selection)
3. `battle-start` -- Battle! (first battle)
4. `battle-start-delayed` -- Choose a Move (chained from step 3)
5. `battle-moves-dismissed` -- Type Advantage (chained from step 4)
6. `effectiveness-dismissed` -- Switch Monsters (chained from step 5)
7. `reward` -- Pick a Reward (after first floor clear)

- Auto-chains battle steps 3-6 with 300-600ms delays
- Skip button available; marked done after all steps shown
- `bm-tutorial-done` localStorage key
